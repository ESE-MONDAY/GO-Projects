# GO-Projects

A collection of small projects I build while learning Go.

## Projects
- `server` — a minimal Go web server. Serves the `static` folder containing `index.html` and `form.html`.

## Run (example)
From the repository root run:

```bash
cd server
go run main.go
```

Open http://localhost:8080 in your browser (or the address shown by the server).

## Notes
These are learning projects; expect iterative changes as I explore Go features.
